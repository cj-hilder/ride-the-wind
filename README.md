# Ride the Wind

A PWA that predicts your bike commute time from the forecast wind and tells you,
the night before, whether to leave early or sleep in.

## Develop
```
npm install
npm run dev
```

## Deploy
Pushing to `main` triggers the GitHub Action, which builds with Vite and
publishes to GitHub Pages at `https://cj-hilder.github.io/ride-the-wind/`.
